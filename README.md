# Concrete Strength Prediction Using Machine Learning

This repository contains a machine learning project designed to predict the compressive strength of concrete based on its mix design, components, and age. 

###  Repository Contents
* **Data & Modeling:** Includes the underlying dataset (`Concrete_Data.xls`) and a Jupyter Notebook (`.ipynb`) detailing the exploratory data analysis and model training process.
* **Web Application:** Features a Python-based web app (`app.py`, `index.html`) that serves the trained model (`model.pkl`) for simple, interactive predictions.
* **Documentation:** Includes a comprehensive analysis report (`.docx`) detailing the project workflow and coding components.

###  Getting Started
To run the web application locally:
1. Ensure you have Python installed along with the necessary data science and web frameworks (e.g., `scikit-learn`, `flask`, `pandas`).
2. Clone the repository and navigate to the project folder.
3. Run `python app.py` in your terminal.
4. Access the web interface via the localhost URL provided in your terminal to test the model.
